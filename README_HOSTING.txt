JL Services Website (Clean Host Package)

Mobile nav:
- Tap "☰ Menu" on mobile to open/close navigation.

Hosting:
- GitHub Pages: Settings → Pages → Deploy from branch → / (root)
- If using GitHub Pages, keep the .nojekyll file.

Local preview:
- Double-click index.html OR (recommended) run a local server:
  python -m http.server 8000
  then open http://localhost:8000
